Dockerfile <https://github.com/vgist/dockerfiles/tree/master/rsync>

Automatically built by Github Actions

#### Volume:

- /mnt

#### Custom usage:

    docker run \
        -d \
        --name rsync \
        -p 873:873 \
        -v /your/path:/mnt \
        -v /your/config:/etc/rsyncd.conf
        gists/rsync

#### Compose example:

    rsync:
      image: gists/rsync
      container_name: rsync
      ports:
        - "873:873"
      volumes:
        - /your/path:/mnt
        - ./rsyncd.conf:/etc/rsyncd.conf
      restart: always
